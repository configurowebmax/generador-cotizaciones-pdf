# Generador de Cotizaciones PDF Gratis

Aplicación web gratuita para **crear cotizaciones profesionales** y exportarlas a PDF con un clic. Ideal para freelancers, consultores y negocios.

## ✨ Características

- Cotizaciones **profesionales** con diseño limpio.
- **Exportación a PDF** con html2pdf.js.
- **Múltiples ítems** con cálculo automático de subtotales y total.
- Datos de empresa **guardados** automáticamente.
- Persistencia local (LocalStorage).
- Diseño Mobile-First optimizado.

## 🚀 Uso

1. Completa los datos de tu empresa y del cliente.
2. Agrega ítems con descripción, cantidad y precio.
3. Haz clic en **"Descargar PDF"**.
